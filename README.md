## How to Upload Your Own Images

If you want to replace the stock photos with your own images, you have **TWO OPTIONS**:

### Option 1: Use the File Editor (Easiest)
1. In your project, click on the "Upload file" button
2. Navigate to the `public` folder
3. Upload your images here
4. Rename them to match the original names (see below), OR ask me to update the code to use your new image names

### Option 2: Ask Me to Update Image URLs
Simply provide me with:
- A link to your image online (Google Drive, Dropbox, etc.)
- OR describe what image you want and I'll find a better stock photo
- Tell me which page(s) the image is for

## Image Optimization Tips

Before uploading, optimize your images:
- Use compressed formats (PNG for logos, JPEG for photos)
- Keep file sizes under 500KB each
- Use tools like TinyPNG or Squoosh.app to compress
- Ensure good quality but not excessively large
